## 1-2-A Claude を使用した記事作成

**手順：**

### Step 1: Claude にプロンプトをコピペ

以下のテンプレートを Claude にコピペします：

```
【目的】
福祉業界・就労支援領域のブログ記事を作成する

【テーマ選択】
10個のカテゴリから日付ベースでローテーション選択
本日（2026-06-01）→ テーマ#2: 「精神障がいを持つ方が働きやすい職場環境とは」

【対象読者】
- 業種: 福祉業界
- 職種: 就労支援スタッフ
- 背景: うつ・精神障がい者の就労サポート経験者

【文体・形式】
- 会話調、やさしい文章（です・ます調）
- 長さ: 600～900文字
- 構成: タイトル → リード文 → 見出し付き本文（2～3セクション） → まとめ
- 専門用語: 最小限、一般読者向け
- トーン: ポジティブ・共感的・温かみのある表現

【禁止事項】
- 特定個人・団体への批判

【出力形式】
タイトル
リード文
## 見出し1
本文...
## 見出し2
本文...
## まとめ
```

### Step 2: Claude が記事を生成

Claude が上記のテンプレートに基づいて、600～900字のブログ記事を生成します。

**出力例：**
```
なぜ「環境整備」が精神障がい者の働きやすさを変えるのか
_
精神障がいを持ちながら働く方の多くは、職場の環境に大きく影響されます。
今回は、どのような職場環境が彼らの成長を支えるのか、一緒に考えてみましょう。
_
## リモートワークと柔軟性の大切さ
...
```

### Step 3: 記事をメモ帳にコピー

生成された記事をテキストエディタ（メモ帳など）に保存しておきます。

## 1-2-B リタリコへの手動転記

### Step 1: リタリコにログイン

1. https://snabi.jp/biz/login にアクセス
2. メールアドレス・パスワードでログイン
   ```
   メール: it-otakanomori@dd-career.com
   パスワード: （.env ファイルで管理）
   ```

### Step 2: ブログ管理画面に移動

1. 左メニューの「ブログ管理」をクリック
2. 「ディーキャリアITエキスパートおおたかの森オフィス」を選択
3. 「新しくブログを書く」をクリック

### Step 3: 記事情報を入力

| 項目 | 入力内容 | 備考 |
|---|---|---|
| **タイトル** | Claude で生成したタイトル | SEO キーワード含める |
| **本文** | Claude で生成した本文 | 見出し構成を保持 |
| **カテゴリ** | 該当するカテゴリを選択 | 複数選択可 |
| **タグ** | 関連キーワード | 例：`精神障がい`、`職場環境` |
| **公開日** | 毎週金曜日 | 自動公開または予約設定 |
| **アイキャッチ画像** | （必要に応じて） | 事業所の写真など |

### Step 4: プレビュー確認

1. 「プレビュー」をクリックして内容確認
2. 誤字脱字、レイアウトをチェック
3. 問題なければ「公開」をクリック

### Step 4: 公開

1. 右上の「保存して公開する」をクリック
2. 「記事一覧」に戻る
3. 記事がリスト上に表示されていることを確認

**リタリコ記事 URL 形式：**
```
https://snabi.jp/biz/facilities/28897/blog_articles/XXXXX
```

**目的：** 事業所の最新ニュース・イベント情報を Google マップ経由でユーザーに届ける

- 明るく、事業所の活気が伝わる写真
- 2～3枚セット で複数角度から
- 100KB～5MB の中サイズが最適

## 5. よくある質問（FAQ）

**Q: X投稿が見つからない場合は？**
A: アカウント ID を確認し、プロフィール画面から「投稿」タブをクリック。管理画面でスケジュール投稿を確認できます。

**Q: ブログが反映されない場合は？**
A: キャッシュクリア（Ctrl+Shift+Del）を実施。1時間待つ。それでも反映されない場合は管理者に連絡してください。

**Q: 口コミに不適切な内容があった場合は？**
A: 報告ボタンをクリックして Google に通報。その後、上長に報告してください。
